# ❤️ Cardio-Predict

An End-to-End Machine Learning and Deep Learning project for predicting cardiovascular disease using patient health records.

---

## 📌 Project Overview

Cardiovascular disease is one of the leading causes of death worldwide. This project uses Machine Learning and Artificial Neural Networks (ANN) to predict whether a patient is at risk of heart disease.

## 📌 Overview

Cardio-Predict is an AI-powered healthcare application that predicts whether a patient is at risk of cardiovascular disease using Machine Learning and Deep Learning models.

The application provides a simple web interface built with Flask where users can enter clinical parameters and receive an instant prediction along with confidence scores.

# 🌐 Running Locally

Clone the repository

```bash
git clone https://github.com/yourusername/Cardio-Predict.git
```

Move into the project directory

```bash
cd Cardio-Predict
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app/app.py
```

Open your browser

```
http://127.0.0.1:5000
```

---

# 🐳 Docker

Build the Docker image

```bash
docker build -t cardio-predict .
```

Run the Docker container

```bash
docker run -p 5000:5000 cardio-predict
```
